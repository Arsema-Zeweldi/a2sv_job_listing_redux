# A2SV Job Listing App (Next.js)

This is a small job-listing web application built with Next.js, TypeScript and Tailwind CSS. It demonstrates listing jobs, a job details view, reusable UI components (cards, icons), and responsive layout.

**What it does**

- **List jobs**: shows a collection of job postings with title, company, location, categories and logo.
- **Job details**: a dedicated job page with description, responsibilities, candidate profile, when & where information and related metadata.
- **Reusable UI**: components such as `JobCard`, `IconBox` and the job detail view which use `react-icons` for consistent iconography.

**Tech Stack**

- **Framework**: `Next.js` (App Router)
- **Language**: `TypeScript`
- **UI**: `Tailwind CSS`
- **Icons**: `react-icons`

**Development**
Start the Next.js development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
